# Two-Wheeler Brand Prediction (Machine Learning Project)

This project predicts the best two-wheeler brand based on features like vehicle type, rating, review sentiment, usage type, service rating, warranty years, and range.

## Technologies Used
- Python
- Scikit-learn
- Pandas, NumPy
- Streamlit
- Joblib
- Synthetic Data Generation

## Features
- End-to-end ML model
- Custom synthetic dataset
- Model evaluation & comparison
- Interactive Streamlit web application

## How to Run
pip install -r requirements.txt
streamlit run app.py
